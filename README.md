LearningPlatform-Flask-MySQL-Stripe
Description
LearningPlatform-Flask-MySQL-Stripe is a web application that provides an online learning platform, allowing users to access and enroll in various courses. The platform is built using Flask, MySQL, and Stripe integration for secure payment processing.
